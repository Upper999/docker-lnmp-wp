# LNMP + WordPress 生产环境一键部署（Docker-Compose）

执行 `./start.sh` 30秒即可得到一个完整可用的 WordPress 个人技术博客！

## 技术栈
- Nginx + PHP8.2-FPM + MySQL8 + Redis
- 全容器化部署
- 数据持久化
- 生产级 Nginx 配置

## 一键启动
```bash
chmod +x start.sh backup.sh
./start.sh
